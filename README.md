# dotfiles.hyprland
Dotfiles for HyprLand

Dependencies:
Waybar,
Mako,
Grim,
Kitty,
Nautilus

Binds: SUPER + S = screenshot

SUPER + C = killactive

SUPER + Q = kitty

SUPER + E = nautilus

SUPER + V = float window
